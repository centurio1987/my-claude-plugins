# interview-simulation

면접 준비를 위한 Claude 플러그인. 질문 생성, 꼬리 질문 생성, 답변 검토 세 가지 스킬과 전체 세션을 오케스트레이션하는 부트스트랩 에이전트를 제공한다.

## 컴포넌트

### 스킬

| 스킬 | 트리거 | 설명 |
|------|--------|------|
| `question-generator` | "질문 생성해줘", "면접 질문 만들어줘" | 포지션별 1회 인터뷰 분량 질문지 생성 (한국어) |
| `followup-generator` | "꼬리 질문 만들어줘", "파생 질문 생성" | 루트 질문 + 답변 기반 꼬리 질문 트리 생성 |
| `answer-reviewer` | "답변 검토해줘", "내 답변 피드백" | 의도 부합도·깊이·톤앤매너 3기준 평가 |

### 에이전트

| 에이전트 | 트리거 | 설명 |
|----------|--------|------|
| `interview-bootstrapper` | "면접 준비 시작", "인터뷰 세션 시작" | 세션 초기화 → 질문 생성 → 인터뷰 진행 → 리뷰 리포트 |

## 사용법

### 전체 면접 세션 (권장)

```
면접 준비 시작해줘
```

`interview-bootstrapper` 에이전트가 포지션 설정부터 최종 리포트까지 전체 흐름을 진행한다.

### 개별 스킬 사용

```
# 질문지만 생성
Node.js 시니어 포지션 면접 질문 만들어줘

# 꼬리 질문 생성
이 답변에 대한 꼬리 질문 만들어줘
[질문과 답변 붙여넣기]

# 답변 검토
내 답변 검토해줘
질문: ...
답변: ...
```

## 이력서 연동

이 플러그인은 `/Users/centurio/resume` 디렉토리에서 이력서를 자동으로 읽어 맞춤형 질문을 생성한다. 해당 경로가 없거나 접근 불가 시 직접 입력을 요청한다.
